# PivotRealEstateGroup.com

This repository contains a marketing landing page that advertises the premium domain PivotRealEstateGroup.com for sale.

## Getting Started

Open `index.html` in any modern web browser to preview the site locally. All styling and interactivity is provided through the accompanying `styles.css` and `script.js` files—no build step is required.

## Structure

- `index.html` – Main landing page markup
- `styles.css` – Global styles, layout, and responsive design
- `script.js` – Lightweight enhancements such as the sticky header animation and dynamic copyright year

Feel free to tailor the content, contact information, or visual design to match your sales strategy.
